

This is my personal portfolio website created as part of the **23CSE113 User Interface Design – Project Phase I**.

The website is developed using **HTML5 and CSS3** and is designed to showcase my profile, skills, and projects in a clean and responsive layo

The portfolio contains the following sections:

- **Home** – Introduction with profile photo and navigation menu
- **About Me** – Brief description about myself and my technical skills
- **Projects** – Showcases my work such as:
  - Dream Team Task
  - Portfolio Website
  - Linux (WSL Setup)
- **Contact** – Clickable email and GitHub profile links

 Features:

- Responsive design using **media queries**
- Hover effects and animations
- Sticky navigation bar
- Interactive project cards
- Profile image with hover glow effect
- Hyperlinked contact section

Technologies Used:

- HTML5
- CSS3
- GitHub Pages (for hosting)
